NoteMe
====
##By Abitalo & Zela
##(2015-11-26)
----
<i>NoteMe is an Android application that help to remind you for your schedule and offer a easy way to take note for things around you.  

(targetSdkVersion 21,minSdkVersion 22)  </i>
